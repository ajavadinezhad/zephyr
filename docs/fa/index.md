features:
  - title: 🕵️ غیرقابل تشخیص توسط DPI
    details: تمام ترافیک از محدوده IPهای گوگل عبور می‌کند و کاملاً شبیه ترافیک عادی گوگل است. هیچ امضای VPN، پورت غیرعادی یا IP سرور اختصاصی برای مسدودسازی وجود ندارد.
  - title: ⚡ ترکیب درخواست‌ها
    details: درخواست‌های همزمان مرورگر در یک فراخوانی Apps Script دسته‌بندی می‌شوند. یک صفحه با ۳۰ درخواست فقط از ۱-۳ اجرا استفاده می‌کند، نه ۳۰ تا.
  - title: 💾 کش داخلی
    details: فایل‌های استاتیک (JS، CSS، فونت، تصاویر) در حافظه ذخیره می‌شوند. پاسخ‌های کش شده کاملاً از رله عبور نمی‌کنند و بارگذاری‌های بعدی را بسیار سریع‌تر می‌کنند.
  - title: 🔄 Failover چند آدرس
    details: امکان تنظیم چندین دیپلوی Apps Script در حساب‌های مختلف گوگل. رله آدرس‌های تنظیم شده را موازی اجرا کرده و از اولین پاسخ موفق استفاده می‌کند.
  - title: 🔒 پشتیبانی کامل HTTPS
    details: پروکسی محلی TLS را termination می‌کند تا سایت‌های HTTPS مسدود شده به صورت شفاف کار کنند. هیچ داده متنی خارج از دستگاه شما ارسال نمی‌شود.
  - title: 📱 VPN اندروید - بدون روت
    details: با یک لمس تمام ترافیک مرورگر از طریق رله در سطح سیستم مسیریابی می‌شود. بدون نیاز به تنظیم هر اپلیکیشن، ADB یا روت.
  - title: 🖥️ دسکتاپ چند پلتفرمی
    details: روی ویندوز، لینوکس و macOS با رابط کاربری ساده یا حالت خط فرمان کار می‌کند. عالی برای توسعه‌دهندگان و کاربران حرفه‌ای.
  - title: 🚀 پروب موازی
    details: ابزار تست دسترسی داخلی بررسی می‌کند کدام نقاط پایانی از شبکه شما کار می‌کنند. قبل از راه‌اندازی بدانید دقیقاً چه چیزی کار می‌کند و چه چیزی نه.
